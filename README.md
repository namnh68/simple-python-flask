# simple-python-flask
A simple python web using Flask

### How to run this app

#### Requirement

- CentOS 7
- Python 2.7
- Pip

### Installation

1. Install PIP

```
yum update -y
yum install -y epel-release
yum install -y python-pip
```

2. Install Flask

```
pip install -r requirements.txt
```

3. Run App

```
python app.py
```
